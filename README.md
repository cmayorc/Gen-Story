# Gen Story

Este es un proyecto Unity para la generación de historias interactivas.

## Descripción

Proyecto Unity que permite crear y gestionar historias interactivas con diferentes personajes y escenarios.

## Tecnologías

- Unity 2022.3.15f1
- C# para la lógica de juego
- URP (Universal Render Pipeline) para renderizado

## Estructura del Proyecto

- **Assets/**: Contiene todos los recursos del proyecto
  - **Scenes/**: Escenas del juego
  - **Settings/**: Configuraciones de renderizado y calidad
  - **MCPForUnity/**: Herramientas de desarrollo MCP para Unity

## Instalación

1. Clona este repositorio
2. Abre el proyecto con Unity 2022.3.15f1 o superior
3. El proyecto se cargará automáticamente

## Uso

1. Abre la escena principal desde `Assets/Scenes/SampleScene.unity`
2. Configura los personajes y escenarios según tus necesidades
3. Ejecuta la escena para probar la historia interactiva
